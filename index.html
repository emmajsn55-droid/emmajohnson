About Me

Flutter Mobile Engineer & Cloud Integration Specialist

I'm a Mobile Engineer specializing in Flutter development and cloud-integrated mobile solutions. I build cross-platform mobile applications that seamlessly connect with cloud backends, delivering exceptional user experiences on iOS and Android.

With expertise in Flutter, Dart, and cloud services (Firebase, AWS Amplify, Azure Mobile Apps, Google Cloud), I create scalable mobile applications that leverage cloud infrastructure for authentication, real-time data synchronization, push notifications, and analytics. I'm passionate about building performant apps with clean architecture, implementing state management solutions (Provider, Riverpod, Bloc), integrating RESTful APIs and GraphQL, and deploying apps to App Store and Google Play. My work focuses on creating responsive UIs, optimizing app performance, implementing offline-first architectures, and ensuring seamless cloud connectivity.
